## What this repo is

`html2md-cli` exposes the `html2md`, `html2md-log-export`, and
`html2md-upload` command-line entry points. This repository contains the
current CLI implementation, including the logic used to fetch/process input
and convert HTML to Markdown, PDF, and TXT outputs.
