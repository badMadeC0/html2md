## 2024-05-19 - Fast-pathing JSON parsing and dictionary construction in python tight loops
**Learning:** In tight parsing loops in python (like large JSONL -> CSV conversions), dictionary comprehensions (`{k: v for k,v in mapping}`) are often faster than manual looping with `.update()` or list comprehensions, because they are implemented directly in C within the CPython interpreter. Attempting to "optimize" away dictionary construction per row by mutating a single dictionary in place actually performed slower when tested, highlighting the importance of benchmarking assumptions.
**Action:** Before optimizing dictionary mutations in Python tight loops, always benchmark the alternatives (dict comprehensions vs explicit loops), as C-level implementations often beat Python-level "avoid allocations" optimizations. Also, remember to clean up performance benchmark scratch files before committing.
