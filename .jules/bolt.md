## 2025-02-17 - Optimize JSONL to CSV conversion in log export
**Learning:** In a high-throughput loop for parsing JSON lines and generating CSV rows, per-row dictionary reconstruction using dictionary comprehensions and function calls for string sanitization adds significant overhead. Pre-computing function references (`json.loads`) and global variables (`_DANGEROUS_PREFIXES`), mutating explicit dictionary objects, and directly passing them to `csv.DictWriter` while leveraging its native `extrasaction='ignore'` and `restval=''` features improves performance significantly (approx ~15-20% time reduction).
**Action:** When handling large file loops involving `json.loads` and `csv.DictWriter`, inline the parsing logic, pre-compute method references in the local scope, avoid creating new dictionaries on every iteration, and leverage the `csv` module's native error handling over explicit dictionary operations like `.get(key, "")`.
